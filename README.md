# 📅 Timetable Generator

A clean, responsive frontend web application designed to help create, manage, and view academic or weekly schedules efficiently. Built entirely with native web technologies.

## ✨ Key Features
* **Interactive UI:** Dynamic schedule table layout designed for readability.
* **Fully Responsive:** Optimised layout that scales smoothly across mobile, tablet, and desktop viewports.
* **Client-Side Build:** Lightweight execution running directly in the browser with no backend dependencies.

## 🛠️ Tech Stack
* **HTML5:** Structuring clean, semantic grid components.
* **CSS3:** Custom styles, layouts, and responsive media queries.
* **JavaScript (ES6):** Dynamic UI interactions and logic management.

## 🚀 How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com
   ```
2. Open `timetable.html` directly in your web browser.
